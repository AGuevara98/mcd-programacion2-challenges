# MCD UDG Challenges

## Overview
This repository contains the implementation of three machine learning challenges:
1. Cancer Detection
2. Glassdoor NLP Analysis
3. Thesis Challenge

## Repository Structure
- `src/common/`: shared modules
- `src/cancer/`: Cancer Detection pipeline
- `src/nlp_glassdoor/`: NLP pipeline
- `src/thesis/`: Thesis pipeline
- `docs/`: technical documentation
- `outputs/`: generated metrics, plots, and models
